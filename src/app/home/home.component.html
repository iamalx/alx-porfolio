Description

Reboot is a non-profit organization that helps veterans transition into civilian life. Reboot is a tablet application created for veterans with the best user experience that makes it effective for veterans to accomplish a set of the task before they leave the military. This includes:

  • A wizard that guides veterans through the app.
  • A timeline that details the task performed and notifications when users are being left behind.
  • And an assessment that shows a veteran's current situation and suggestions on areas that need improvement.

  • Used JavaScript, Angular an Ionic to code a prototype application.
  • Designed RESTful backend server allowing personal data to be stored persistently in an online database.
